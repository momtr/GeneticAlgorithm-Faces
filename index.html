Faces - Genetic Algorithm

Wie funktioniert's?

Das Modell basiert auf einem sogenanntem "Genetic Algorithm" (deutsch: genetischer Algorithmus), was Gemeinsamkeiten zu der tatsächlichen Evolution hat.

Der Algorithmus basiert auf folgenden Prinzipien:
  • Vererbung (Weitergabe genetischer Merkmale
  • Selektion (Selektion der am besten geeigneten Kreaturen)
  • Variation (Variation von Merkmalen)

Genotyp vs. Phenotyp

  • Der Genotyp ist die Abfolge bestimmter Erbgutinformationen (in der DNA zu finden).
  • Der Phenoty ist die visuelle Ausprägung dieser Merkmale.
Beispielsweise:
Genotyp: A,B,C,D,A,B,B,B,C,D,D,A,D,C
Phenotyp: eine rote Blume mit gelben Blüten

Im Allgemeinen

      1. Erschaffen Sie eine Population von n Elementen 
      (welche Kreaturen oder Ähnliches sein können)
      2. Wiederholen Sie die nächsten Schritte vortlaufend:
          2a. Ermitteln Sie die zwei Elemente der Population,
           welche den besten Ansatz zur Problemlösung bereitstellen
          2b. Kreuzen Sie zwei zufällige Elemente der Population 
          um daraus zwei Kinder zu bekommen
          2c. Übertragen Sie die beiden Elemente auf die gesamte Population
          2d. Mutieren Sie die gesamte Population
          3e. erhöhen Sie die Population
